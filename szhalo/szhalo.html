Alapparancsok

Interfész konfiguráció

konzol parancs leírás
Router# enable Belépés privilegizált módba
Router# configure terminal Belépés konfigurációs módba
Router(config)# interface fa 0/0
Router(config)# interface ser 2/0
Router(config-int)# ip address 192.168.0.1 255.255.255.0 IP cím megadása
Router(config-int)# clock rate 6400 Órajel beállítása (csak seral kábel esetén a DCE - órajeles)
Router(config-int)# no shutdown Interfész felkapcsolása
Router(config-int)# exit Visszalépés egy szintet

Forgalomirányítási beállítások

konzol parancs leírás
Router(config)# ip route [IP cím] [alhálózati maszk] [next hop] Statikus útvonal konfigurálása (cél hálózat/cél hálózat maszk/következő router címe)
Router(config)# ip route 0.0.0.0 0.0.0.0 [next hop] Alapértelmezett útvonal konfigurációja
Router(config)# router rip Belépés RIP konfigurációs módba
Router(config-router)# version 2 RIP verziószám megadása
Router(config-router)# network 192.168.2.0 RIP hálózatok megadása (amelyek ehhez a routerhez csatlakoznak)
Router(config-router)# default-information originate RIP alapértelmezett útvonalak hirdetése
Router(config-router)# redistribute static RIP Statikus útvonal hirdetése
Router(config-router)# no auto-summary RIP - CIDR: classless címek hirdetése azonos/változó alhálózatokra bontás esetén kötelező!
Router(config-router)# passive-interface FastEthernet 4/0 RIP frissítések letiltása megadott interfészen

Beállítások vizsgálata

konzol parancs leírás
Router# show ip route Forgalomirányítási tábla lekérése
Router# show startup-config Indítási konfiguráció lekérdezése
Router# show running-config Futó konfiguráció lekérdezése

VLAN

VLAN konfigurálása Router-en

konzol parancs leírás
Router(config)# interface FastEthernet X/Y
Router(config-if)# no shutdown
Router(config-if)# exit
Router(config)# interface FastEthernet X/Y.Z
Router(config-subif)# encapsulation dot1Q [VLAN]
Router(config-subif)# ip address [IP cím] [alhálózati maszk]
Router(config-subif)# no shutdown
Router(config-subif)# exit
  • X,Y - Az adott interfész azonosítója
  • Z - Az adott al-interfész azonosítója (általában a VLANID-ja)
  • [VLAN] - A VLAN azonosítója (ID)

VLAN konfigurálása Swich-en

konzol parancs leírás
Switch> enable
Switch# configure terminal
Switch(config)# vlan [VLAN]
Switch(config-vlan)# name [VLAN_NEVE]
Switch(config-vlan)# exit
  • [VLAN] - A VLAN azonosítója (ID)
  • [VLAN_NEVE] - A VLAN Beszédes neve

VLAN Access konfigurálása Swich-en

konzol parancs leírás
Switch(config)# interface FastEthernet X/Y Egy interfészre
Switch(config)# interface range FastEthernet X/Y-Z Interfész tartományra
Switch(config-if-range)# switchport access vlan [VLAN]
Switch(config-if-range)# no shutdown
Switch(config-if-range)# exit
  • X,Y - A kezdeti interfész azonosítója
  • Z - Az utolsó interfész azonosítója
  • [VLAN] - A VLAN azonosítója (ID)

VLAN Trunk konfigurálása Swich-en

konzol parancs leírás
Switch(config)# interface FastEthernet X/Y Egy interfészre
Switch(config)# interface range FastEthernet X/Y -Z Interfész tartományra
Switch(config-if-range)# switchport mode trunk
Switch(config-if-range)# switchport trunk allowed vlan add [VLAN 1]
Switch(config-if-range)# switchport trunk allowed vlan add [VLAN 2]
Switch(config-if-range)# switchport trunk allowed vlan add [VLAN N]
  • X,Y - A kezdeti interfész azonosítója
  • Z - Az utolsó interfész azonosítója
  • [VLAN1|2|N] - A VLAN azonosítója (ID)
